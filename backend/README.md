# Backend curricular (demo)

Este directorio incluye un ejemplo de script en Python para:

- Consultar páginas del Currículum Nacional de Mineduc.
- Extraer información básica de algunos OA.
- Generar un archivo `data/oa-data.json` que luego es consumido por el front con `fetch("data/oa-data.json")`.

> Nota: este script es solo una base. Debes ejecutarlo en tu propio PC (con conexión a Internet) y ajustar
> las URLs, selectores y lógica según los OA y asignaturas que quieras automatizar.
